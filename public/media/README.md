Put your generated assets here.

- `media/img/logo.png` → copied to `public/media/img/logo.png` (used by the navbar favicon + logo).
- `media/hero/*` → hero preview images (landscape recommended).
- `media/gallery/*` → showcase gallery images (landscape required for uniform tiles).

Run `npm run sync:media` (or just `npm run dev`, it runs automatically before builds) after adding new files.
